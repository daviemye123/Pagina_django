Llamar variables de un contexto (como una lista)

{% if fruits %} {% for x in fruits %}

Fruta: **{{ x }}**

{% endfor %} {% else %}

No hay frutas para mostrar.

{% endif %}

Crear y usar una variable temporal dentro de 'with'

{% with firstname="Tobias" %}

Hola **{{ firstname }}**, ¿cómo estás?

{% endwith %}

**{{ firstname }}** (Este valor estará vacío ya que 'firstname' solo existe dentro del 'with' anterior).

Iterar sobre objetos dentro de una lista

{% if mymembers %}
    {% for member in mymembers %}
  • Nombre: **{{ member.firstname }}**
    • {% endfor %}
{% else %}

No hay miembros para mostrar.

{% endif %}
